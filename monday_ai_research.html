The Three Pillars (Post-May 6, 2026)

On May 6, 2026, monday.com announced what Eran Zinman called "the biggest change in the history of our company" — a full repositioning from "work management platform" to "AI Work Platform." The marketing surface was simplified from ~10 AI brand names down to three native products plus supporting infrastructure.

Pillar 1 · assistant

monday sidekick

"Built-in personal AI assistant" — context-aware, in-product. Knows your role, your work, your account. Sends emails, schedules meetings, generates images, summarizes ticket history, monitors SLAs, builds roadmaps. 500K+ messages processed pre-launch (Q4 2025).

Pillar 2 · vibe coding

monday vibe

"Build any business software in minutes." AI app builder — describe an app in plain English, get a working presentation, inventory app, recruiting app, event portal. Built on mondayDB. Fastest product to $1M ARR in monday's history (Q4 2025).

Pillar 3 · autonomous · Early Access

monday agents

Autonomous AI agents that "act without time, volume, or language constraints." Pre-built across Operations, Project, Marketing, Sales, Engineering. Custom builder with Knowledge + Integrations + Actions + Guardrails. End-to-end workflows.

Cross-cutting platform: monday workflows (agentic visual automation, replaces legacy automation builder) · monday MCP (Claude/Cursor/ChatGPT/Copilot Studio one-click connectors) · AI Platform Gateway (multi-LLM access to Claude/GPT/Gemini) · Agentalent.ai (hiring marketplace for external agents) · AI Trust Center (governance).

02

Brand Evolution — What Was Renamed, What Stayed

monday's AI naming has shifted significantly from the 2024 "AI Blocks + magic + Digital Workforce" launch to the 2026 "Sidekick + Vibe + Agents" reorg. Some products were renamed; some were absorbed; some were quietly deprecated.

2024–early 2025 (legacy branding)

  • AI Blocks — modular AI actions in automation
  • monday magic — describe a workflow, AI builds the board
  • AI Hub — discovery surface
  • Digital Workforce / Digital Workers — autonomous agents
  • SDR Expert agent · AI Service Agent
  • Autopilot Hub
  • AI Service copilot
→

Post May 6, 2026 (current branding)

  • monday sidekick (= old assistant + magic-style chat)
  • monday vibe (= app builder, new product line)
  • monday agents (= old Digital Workforce, unified)
  • monday workflows (= legacy automation builder + agents, unified)
  • monday MCP (= new — external AI clients)
  • AI Platform Gateway (= multi-LLM access layer)
  • Agentalent.ai (= external agent marketplace from monday agent labs)

Important infrastructure consolidation (Jan 26, 2026): monday explicitly deprecated the legacy "Integration for sentence builder" feature. Per the developer changelog: "monday is consolidating all automation feature creation into a single workflows-based infrastructure powering the new automation builder, monday workflows, and monday sidekick." Apps using the legacy API must migrate by April 30, 2026 to remain available in new automation experiences. Existing automations keep running during the transition; a self-service migration wizard is provided in the Developer Center.

monday AI Formula

monday AI= AI Platform Gateway// multi-LLM router — Claude, GPT, Gemini
+ mondayDB foundation// the structured data layer monday vibe builds apps on
+ Unified workspace context// Work Management + CRM + Dev + Service in one
+ Infinite knowledge// "AI learns from documents, workflows, historical data"

+ AI Credits consumption// $0.01 per credit — usage-based pricing on top of seats

Architectural difference from competitors: Where Linear is narrow + deep (dev workflows), ClickUp is wide + deep (work OS), and Jira is "Rovo + Atlassian Intelligence" (two siloed AI layers), monday's bet is multi-product breadth — the same AI fabric (Sidekick + Agents + Vibe + Blocks) is woven through Work Management, CRM, Dev, and Service. The AI Platform Gateway is the shared abstraction that lets the same prompt route to Claude or GPT or Gemini depending on workload.

The successor to the old "automation builder." monday workflows is the infrastructure that powers the new automation builder, monday agents, AND monday sidekick — all on one substrate.

Quote (positioning): "Autonomous workflows that run work end to end." This is monday's framing for what other vendors call "agent flows" (Atlassian Automation), "Skills" (Linear), or "Autopilot" (ClickUp).

monday MCP — Connect Any AI Assistant

monday MCP — official OAuth-gated server, all plans, no extra cost

Server URL: https://mcp.monday.com/mcp

Marketplace install: Admins install the monday MCP app from the marketplace to enable account-wide access.

Auth: Per-user OAuth — each user authorises individually. "MCP only has access to the data that you, as an individual user, can access."

Plan availability: All plans, currently no additional cost for monday MCP itself. (Costs can come from the LLM client side.)

Two modes: platform MCP (workspace automation) and apps MCP (developer mode for app-building, via @mondaydotcomorg/monday-api-mcp with --mode apps flag).

Supported AI clients (one-click connectors)

Claude (Anthropic)

monday.com is on Claude's official connector directory — Pro / Max / Team / Enterprise plans, web + desktop + mobile.

ChatGPT (OpenAI)

Requires ChatGPT Plus or Pro. Configured via Settings → Connectors → Advanced (developer mode).

Cursor

monday plugin is verified by Cursor — installs the MCP server automatically.

Microsoft Copilot Studio

Add monday MCP server as a tool to any agent you build in Copilot Studio using the native connector. Also supports Mistral Le Chat and any other MCP-compatible tool.

What monday MCP exposes (tools)

Boards (create, read, modify) Items (CRUD) Columns (create, update, manage) Updates / comments Users / teams Workspaces Notetaker meetings query (Apr 2026 API) knowledge_base_search (AI search across snippets) ask_developer_docs (Apr 2026 API) Project / sprint queries (dev product) CRM operations (leads, deals, pipelines)

Use cases monday publishes: "generate sprint summaries, team performance reports, deadline tracking" · "convert meeting notes into structured items, assign tasks, update statuses through natural language" · "ask for rollups like 'What's blocking the launch?' across Product, Marketing, and RevOps boards" · "create and triage incidents, support requests" · "create leads and deals, update pipeline stages."

AI Platform Gateway — Multi-LLM Access

Announced as part of the May 6, 2026 AI Work Platform launch. The AI Platform Gateway is the abstraction layer that lets monday route the same prompt to Claude, GPT, or Gemini transparently — both for internal features (Sidekick uses different models for different intents) and for customer-built agents.

"Additional capabilities include access to multiple large language models through monday's AI Platform Gateway, new AI-powered development tools in monday vibe, and a redesigned mobile app that brings Sidekick and agents together in one place." — IR press release, May 6, 2026

Models confirmed (per /w/ai)

Claude (Anthropic) GPT (OpenAI) Gemini (Google) "and more"

Where this differs from ClickUp's multi-model: ClickUp lets users toggle the model per prompt on Everything AI ($28). monday's Gateway is largely opaque at the user surface — the platform routes based on intent. Customers who want explicit model choice get it through monday MCP + their own client (Claude.ai, ChatGPT, Cursor), not through Sidekick's UI.

Agentalent.ai — The AI Agent Hiring Marketplace

Launched Mar 23, 2026 · monday agent labs

"A managed marketplace where enterprises can discover, evaluate, and hire AI agents for defined business roles."

Agentalent.ai is the first launch from monday agent labs, monday.com's incubation engine for agentic work. It's a two-sided marketplace — enterprises post agent "roles," developers/builders publish agents, monday authenticates, authorises, and qualifies them. Pricing starts around $2,000/month per agent (per Constellation Research).

How it works (both sides)

For enterprises

  • Post a role description (like a job posting)
  • Browse qualified agents matching the role
  • Evaluate based on task fit, business requirements, operational readiness
  • Test performance before adoption
  • Onboard agent alongside human team using existing HR-like process

For developers / agent builders

  • Onboarding into Agentalent — authentication, authorisation, qualification
  • Contract management + billing handled by monday
  • Qualification process gates agents before exposure to enterprises
  • Distribution channel — reach "large companies" via the marketplace

Launch partners + early customers

  • Built with: AWS + Anthropic. "Leveraging the most advanced frontier models, which handle highly complex agent workloads and are built specifically for the enterprise."
  • Early enterprise customers: Wix, Mesh Payments
  • Channel partners exploring agent roles: Matrix, Ness Xebia, Devoteam, Impresoft Engage, Demicon — across marketing, campaign execution, operational workflows
  • 17 agents available at launch (per Constellation Research analysis)
"Every company will soon have a blended workforce of humans and AI agents… Agentalent.ai helps companies define roles, evaluate capability, and onboard AI agents alongside human teams using processes they already understand. As these agents become increasingly capable of operational work, I invite every company to open its ideal AI position and let us help find the right match." — Roy Mann, co-founder & co-CEO · Mar 23, 2026

Strategic significance: This is the most commercially aggressive move in the PM-tool AI space — monday positions itself not just as the platform that hosts agents but as the employment agency for AI workers. No competitor has a marketplace at this layer. ClickUp talks about "Certified Agents" without a productised marketplace yet. Atlassian has Forge Marketplace for apps, not agent-as-worker. Constellation: "It's quite possible that enterprise software vendors will start to resemble AI employment agencies."

AI Credits Pricing Model

Pricing event: Starting May 6, 2026 — the same day as the "All In on AI" announcement — new customers must purchase AI credits alongside seats. Sidekick credit consumption activates May 20, 2026; Agents consumption activates June 8, 2026. Existing customers can opt in. Enterprise customers receive complementary AI packages.

The credit unit

  • 1 AI Credit = $0.01
  • AI Sidekick conversations (under the cap) and monday magic workspace generation do NOT consume credits
  • Free / no-credit features: Formula Builder, Docs Assistant, Deal Insights, Risk Insights

What consumes credits

  • monday sidekick — per message over the daily cap

Community pricing criticism (Mondaywiki, Nubia Magazine, TaskRhino): "AI credits run out faster than expected" · "per-user AI credit limits introduced in 2026 mean heavy users frequently hit a wall" · "the change to non-renewable free AI credits, a new usage model where actions use 8 times as many AI credits and super expensive 'starter' AI Add-on Pricing Plans." Pricing friction is monday's biggest user-side complaint — similar to ClickUp's "AI as add-on" issue but more granular and more painful for power users.

Strategic Vector — "AI Work Platform" + Digital Workforce

May 6, 2026 · platform repositioning

monday.com is no longer a "work management platform" — it's an "AI Work Platform"

This is monday's most strategic move of the year. The repositioning is not cosmetic: the product surface was consolidated, the pricing model was rebuilt around AI credits, and the company built an external marketplace (Agentalent) for AI labor. monday is staking out the "AI employment agency for the enterprise" position — a category no competitor has fully claimed.

"We have re-architectured the core of our platform around a single belief that work should be orchestrated between humans and AI agents at scale from a single system of record." — Eran Zinman, co-founder & co-CEO

3 / 6 / 12-month outlook

Next 3 months (May → Aug 2026)

  • monday agents → GA (out of Early Access)
  • AI Credits go fully live on Sidekick (May 20) and Agents (Jun 8)
  • Agentalent.ai expands beyond launch 17 agents
  • Mobile app + Sidekick + agents unified UX rollout
  • Legacy "Integration for sentence builder" deprecation deadline: Apr 30, 2026 (already passed for new automations)

Next 6 months

  • Service Agent 2.0 GA
  • Skills Marketplace ecosystem matures — third-party developers monetising Sidekick Skills
  • Project-level AI Blocks (cross-board / portfolio scope)
  • Resource allocation AI with skills + availability matching
  • AI Campaigns expansion in monday CRM

Next 12 months

  • Risk management AI across all PM projects
  • Full resource allocation AI (effort + availability + skills)
  • Agentalent matures — agent marketplace as a real revenue line
  • monday positions as the SMB/mid-market "digital staff augmentation layer"
  • Potential ISO 42001 push for enterprise AI governance

The monday vector summary: SMB Digital Workforce + vibe coding democratization + agent labor market. Three distinct bets, all aligned: (1) Sidekick = assistant for every seat, (2) Vibe = no-code AI app builder for non-devs, (3) Agents + Agentalent = supply the AI workforce itself. monday is the only PM vendor playing all three. The risk: complexity in their own surface area, credit-pricing churn, and reliance on Claude/GPT/Gemini for the actual reasoning under the hood.
